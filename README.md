# RAG Using Qdrand and Gemini Example

This repo shows the example to build a Retrieval Augmented System (RAG) using `Google Gemini`, `Qdrant` and `llama_index` with `Streamlit` user interface.

## Dependencies.

```
database = "Qdrant"
foundament_model = "models/gemini-2.0-flash"
embed_model = "models/gemini-embedding-exp-03-07"
```
